# Betting Baseline (NBA) — 6-Month Feature Roadmap

## Month 1: Game Day

- **Today's slate** — All NBA games with model total, moneyline probability, and edge vs. DraftKings.
- **Rest advantage indicator** — Clear badge when one team is on B2B or has significant rest advantage.
- **Injury report integration** — Pull NBA injury report and flag games with All-Star absences.
- **Referee foul tendency note** — Flag games assigned to high/low-foul referees.

## Month 2: Team Analytics

- **Pace & efficiency dashboard** — ORTG, DRTG, pace, net rating in a sortable table.
- **Home/away split tracker** — Teams that significantly outperform at home.
- **Winning streak / cold streak badge** — Current form indicator on matchup cards.

## Month 3: Player Props

- **Points/assists/rebounds prop calculator** — Enter DraftKings line; see model probability and edge.
- **Matchup edge analysis** — Flag props where the opposing team's defensive rank is weak vs. the player's primary stat.
- **Hot streak indicator** — Players averaging 20%+ above season average in last 5 games.

## Month 4: Historical Analysis

- **Season accuracy report** — Totals and moneyline accuracy by month and team.
- **Best/worst model matchups** — Teams the model consistently over/under-rates.
- **Referee stats** — Historical scoring totals by referee assignment.

## Month 5: Betting Tools

- **Parlay builder** — Select 2–5 games; compute combined probability.
- **CLV tracker** — Opening vs. closing DraftKings lines for this week.
- **Kelly bet size calculator** — Enter bankroll; see recommended stake for each pick.

## Month 6: Automation

- **Morning email** — Day-of NBA picks sent at 9 AM ET.
- **Nightly data refresh** — GitHub Action updates hoopr stats and injury report.
- **Discord picks** — Post daily top pick and value bet to Discord.
